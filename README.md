# Math Quiz Game (C++)

A console-based math quiz game written in C++.

## Description
This program generates random math questions for the player.  
The user can choose:

- Number of questions
- Difficulty level (Easy, Medium, Hard, Mix)
- Type of operation (+, -, *, / or Mix)

The program checks the answers and shows the final result at the end of the game.

## Features
- Random math questions
- Multiple difficulty levels
- Random operators
- Score tracking
- Option to replay the game

## Language
C++
